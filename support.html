📖 What is NetRoute?

NetRoute is a powerful network routing and proxy management tool for iOS. It allows you to configure and manage proxy connections with advanced routing rules, giving you complete control over how your device connects to the internet.

Key Features:

  • Multiple proxy profile management (SOCKS5, HTTP, HTTPS, Shadowsocks)
  • Advanced domain-based routing rules
  • Real-time connection monitoring and testing
  • Network activity logging
  • Customizable interface with multiple themes
  • Complete privacy - all data stored locally on your device

❓ Frequently Asked Questions

How do I add a proxy profile?

1. Open NetRoute
2. Tap the Profiles tab
3. Tap the + button
4. Enter your proxy details (host, port, protocol)
5. Tap Save

What proxy protocols are supported?

NetRoute supports:

  • SOCKS5 (recommended for most use cases)
  • HTTP
  • HTTPS
  • Shadowsocks

How do routing rules work?

Routing rules allow you to specify which domains should use the proxy and which should connect directly. For example, you can route example.com through a proxy while letting other traffic connect normally.

Does NetRoute provide proxy servers?

No. NetRoute is a configuration and management tool. You must provide your own proxy servers. The app does not provide, operate, or sell proxy services.

Is my data safe?

Yes! NetRoute does not collect any data. All your proxy configurations, routing rules, and settings are stored locally on your device only. We never transmit your data to any servers.

How do I test my connection?

1. Go to the Dashboard
2. Connect to a profile
3. The connectivity test will run automatically
4. View latency and connection status

Can I use NetRoute in the background?

Yes! NetRoute supports background location updates for tracking your network connectivity. You can enable "Always" location permission in Settings to allow the app to monitor your connection even when not actively using it.

🔧 Troubleshooting

Connection Not Working

  • Verify your proxy server details (host, port, credentials)
  • Check that your proxy server is online and accessible
  • Try testing the connection from the Dashboard
  • Review the Logs tab for error messages

App Crashes or Freezes

  • Force quit the app and restart
  • Check for app updates in the App Store
  • Restart your device
  • If the issue persists, contact support

📱 System Requirements

  • iOS 17.0 or later
  • iPhone, iPad, or iPod touch
  • Internet connection
  • Access to proxy server credentials (if using authentication)
